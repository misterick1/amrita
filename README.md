## 👁 Технологический Стек контура "Еженышь"

Проект переведен из пассивной prompt-матрицы в полностью исполняемый код:

1. **Всевидящее Око Бабаты (`ezhenysh_bot.py`)** — ИИ-интерфейс в Телеграм со зрением (OCR Pytesseract), который сканирует скриншоты реальности (уведомления с `pump.fun`, `TikTok`, `Trust Wallet`, `Pi Network`), вычисляет скрытые частоты и автоматически отсекает деструктивные паттерны нижних чакр ("Игры в Кальмара").
2. **Мем-Синхронизатор (`swarm_meme_core.py`)** — Ядро, препарирующее импульсивный хайп красного спектра (моггинг, мемкоины типа `$ANSEM`/`mogsem`) по шкале экологичности Амриты.
3. **Блокчейн-Верификатор (`solana_qnt_token.rs`)** — Неизменяемый смарт-контракт на Rust (Solana/Anchor), фиксирующий жесткую эмиссию в 108 QNT монет без инфляции.
4. **Каузальный Трекер Памяти** — Локальный модуль контроля физических параметров, защищающий устройство от лимитов дискового пространства и ведущий вечный лог в `history_log.json`.

## 🛠 Быстрый запуск Всевидящего Ока

```bash
# 1. Установите зависимости контура
pip install pyTelegramBotAPI solana solders pillow pytesseract

# 2. Установите системный парсер зрения (Tesseract OCR)
sudo apt-get install tesseract-ocr tesseract-ocr-rus tesseract-ocr-eng

# 3. Запустите Еженыша в эфир
export TELEGRAM_BOT_TOKEN="your_token_here"
python ezhenysh_bot.py
```
